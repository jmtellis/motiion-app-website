Do's and Don'ts

✓ Do

  • ✓ Let the dark canvas and talent media carry the color; keep chrome neutral.
  • ✓ Use white (primary) for the main action; reserve accent teal for glows, verified badge, and focus.
  • ✓ Express depth with hairline borders and one-step-lighter surfaces.
  • ✓ Use Geist Mono for technical metadata and keyboard shortcuts.
  • ✓ Make focus states unmistakable — the product is keyboard-first.
  • ✓ Keep to the 4px spacing scale and defined radius steps.

✗ Don't

  • ✗ Look like a job board or listing site — no dense generic listings.
  • ✗ Use soft grey drop-shadows, glossy gradients, or material elevation.
  • ✗ Flood the UI with color, emoji, or multiple accents.
  • ✗ Introduce a serif, a third sans, or essential text below body-sm.
  • ✗ Fill primary buttons with accent by default; amber is emphasis.
  • ✗ Improvise off-scale spacing or radii for one-off layouts.
